主要功能:

  • 可添加VOD·CMS·IPTV进行影视播放
  • 支持阿里云盘转存
    添加阿里云盘后需在aliyun中长按阿里云盘设置Token后才可进行转存
    格式为：refresh_token 32位
  • 云盘空间不足时可打开阿里云盘在右上角一键删除缓存

TV操作:

  • 详情页，长按标题可设置关注、刷新海报（如果有更新）、将剧集清单列表翻转 转存源中可以删除转存记录
  • 设置页面，长按版本（ - 可开启调试，更换端口，清除缓存
  • 可以通过长按VOD或CMS一键删除对应的所有片源

VODS 支持搜索

  • 支持http后缀为.json的链接订阅
    如调试链接:
  • 支持导入本地json文件
  • 支持tvbox源数据里的格式一致的源，type=3
    (题外话：不要尝试添加其他VOD源，因为基本没有适配)
  • 源格式如下(可直接复制进订阅中使用):
    {"key":"csp_Zxzj","name":"Zxzj",
    "type":3,"api":"csp_Zxzj",
    "searchable":1,"quickSearch":1,
    "changeable":1,
    "ext":"https://www.zxzj.pro/"}
  • 常见问题:
    • VOD源的卡顿或无法播放为正常现象
      可能是人流原因也可能是网络环境问题
      习惯就好
    • 加不了VOD源的原因可能是源的格式有问题
    • VOD源里面是空白的原因 可能是源没有适配也可能是网络环境问题
    • 这类源可能会失效 不要太过惊讶

CMS 支持搜索

  • 支持http后缀为.json的链接订阅 可与VOD源放在一起
    如调试链接:
  • 支持导入本地json文件
  • 支持tvbox源数据里的格式一致的源，type=1
  • 源格式如下(可直接复制进订阅中使用):
    {"key":"yhys","name":"yhys","type":1,
    "api":"https://m3u8.apiyhzy.com/
    api.php/provide/vod/",
    "searchable":1,"quickSearch":1}
  • 常见问题:
    • CMS源的卡顿或无法播放为正常现象
      可能是网络环境问题但大概率是源失效了
      总而言之习惯就好
    • 加不了CMS源的原因可能是源的格式有问题
    • CMS源里面是空白的原因
      大概率是源失效了也可能是网络环境问题
    • 这类源大概率会失效 不要太过惊讶
